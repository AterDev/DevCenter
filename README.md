# DevCenter
一个针对开发团队内部使用的资源导航系统

# 特性

## 资源
资源定义，包含
- 名称
- 类型
- 所属环境
- 所属项目(多个)
  
## 资源类型
定义资源类型，每个类型，要有图标和颜色，如
- 主机连接
- 数据库连接
- 导航
- 账号
- 应用连接
- 配置，如json/yml/xml格式内容

## 资源属性
元素包含的属性定义，包含
- 显示名
- 唯一名称
- 值类型,long/string/bool/double/datetime/json
- 值
- 是否有效


## 资源组
一组资源的集合，包含
-  名称
-  所属环境
